# 🤖 AI Email Classifier using n8n

## 📌 Project Overview
This project is an automated email classification system built using n8n and OpenAI API.

It automatically:
- Reads incoming emails
- Classifies them into categories (Sales, Lead, Complaint, Spam, Internal)
- Applies corresponding Gmail labels

---

## ⚙️ Tech Stack
- n8n (Workflow Automation)
- OpenAI API (Email Classification)
- Gmail API (Label Automation)

---

## 🔄 Workflow
1. Gmail Trigger → Fetch email
2. Process email content
3. Send data to OpenAI API
4. Classify email category
5. Route using Switch Node
6. Apply Gmail Label automatically

- Fully automated email sorting
- AI-powered classification
